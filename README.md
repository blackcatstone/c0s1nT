1.가상환경에서 main.py 실행

3.로컬에서 tor.py 코드에 디스코드봇 초대 디스코드 디벨롭 사이트 접속 후 봇생성 후 토큰 발급 코드에 your token 부분에 토큰 입력, 서버 채널id, 채널 id  입력 
4.database2.py 코드 열람 하여 최초 db 입력하도록 실행 
5.디스코드 명령어 !t,!h~!h3 입력 자유롭게 사용
